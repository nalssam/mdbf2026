# 역사의 문을 열어라 — 6·25 전쟁 방탈출 게임 🗝

6·25 전쟁의 **원인과 경과**를 방탈출 형식으로 학습하는 웹 게임입니다.
설치나 서버 없이 `index.html` 파일 하나로 동작합니다.

## 실행 방법

- `index.html` 파일을 더블 클릭해서 브라우저(크롬, 엣지 등)로 열면 바로 시작됩니다.
- 인터넷 연결이 없어도 동작하므로 교실 PC·태블릿에서 그대로 사용할 수 있습니다.

## 게임 구성

박물관의 '6·25 전쟁 특별 전시실'에 갇힌 플레이어가 다섯 개의 방을 차례로 통과하며 탈출합니다.

| 방 | 학습 주제 | 퍼즐 |
|---|---|---|
| 1. 기록 보관실 | 전쟁의 배경과 원인 (광복, 38도선 분단, 남북 정부 수립, 애치슨 선언) | 문서 해독 퀴즈 → 금고 비밀번호 `3848` |
| 2. 새벽의 사이렌 | 전쟁 발발 (기습 남침, 서울 함락, 유엔군 참전, 낙동강 방어선) | 사건 카드 시간 순서 맞추기 → 자물쇠 `0625` |
| 3. 작전 상황실 | 인천 상륙 작전 (맥아더, 서울 수복, 압록강 북진) | 지도에서 상륙 지점 찾기 → 자물쇠 `0915` |
| 4. 얼어붙은 겨울 | 중국군 개입과 후퇴 (흥남 철수, 1·4 후퇴, 전선 교착) | 사건–설명 짝 맞추기 → 자물쇠 `0104` |
| 5. 협상 테이블 | 정전 협정과 전쟁의 결과 (휴전선, 이산가족, 휴전 상태) | O/X 퀴즈 5문제 → 탈출 자물쇠 `0727` |

## 학습 지원 기능

- **📓 학습 노트** — 퍼즐을 풀 때마다 핵심 내용이 자동으로 정리됩니다.
- **💡 힌트 시스템** — 방마다 2단계 힌트를 제공하며, 사용 횟수가 기록됩니다.
- **⏱ 타이머** — 탈출까지 걸린 시간이 기록되어 모둠별 대결 활동에 활용할 수 있습니다.
- **탈출 후 정리** — 엔딩 화면에서 전쟁의 전체 흐름을 타임라인으로 다시 정리합니다.

## 수업 활용 팁

- 모둠별로 태블릿 1대씩 나누어 탈출 시간을 겨루는 협동 활동으로 활용해 보세요.
- 힌트 사용 횟수를 감점 요소로 활용하면 토론이 더 활발해집니다.
- 탈출 후 엔딩 화면의 타임라인을 보며 전체 흐름을 함께 정리하는 것을 추천합니다.
