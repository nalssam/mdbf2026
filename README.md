# 역사의 문을 열어라 — 6·25 전쟁 스토리텔링 방탈출 🗝

6·25 전쟁의 **원인과 경과**를 몰입형 스토리텔링 방탈출로 학습하는 웹 게임입니다.
`index.html` 하나로 동작하며, AI(Higgsfield)로 제작한 삽화·배경음악·효과음·이야기 영상이 포함되어 있습니다.

## 실행 방법

- 저장소를 내려받아 `index.html`을 브라우저(크롬, 엣지 등)로 열면 바로 시작됩니다.
- `assets/` 폴더까지 함께 있으면 인터넷 없이도 동작합니다.
- Vercel에 연결되어 있다면 배포 주소로 바로 접속할 수 있습니다.

## 게임 구성

박물관 '6·25 전쟁 특별 전시실'에 갇힌 플레이어가 다섯 개의 방을 통과하며 탈출합니다.
방마다 **삽화 배경 위에서 한 줄씩 눌러 읽는 스토리**가 흐른 뒤 퍼즐이 열립니다.

| 방 | 학습 주제 | 퍼즐 |
|---|---|---|
| 1. 기록 보관실 | 전쟁의 배경·원인 (광복, 38도선 분단, 남북 정부 수립, 애치슨 선언) | 문서 해독 퀴즈 → 금고 `3848` |
| 2. 새벽의 사이렌 | 전쟁 발발 (기습 남침, 서울 함락, 유엔군 참전, 낙동강 방어선) | 사건 카드 시간 순서 맞추기 → `0625` |
| 3. 작전 상황실 | 인천 상륙 작전 (맥아더, 서울 수복, 압록강 북진) | 지도에서 상륙 지점 찾기 → `0915` |
| 4. 얼어붙은 겨울 | 중국군 개입과 후퇴 (흥남 철수, 1·4 후퇴, 전선 교착) | 사건–설명 짝 맞추기 → `0104` |
| 5. 협상 테이블 | 정전 협정과 전쟁의 결과 (휴전선, 이산가족, 휴전 상태) | O/X 퀴즈 5문제 → 탈출 자물쇠 `0727` |

## 몰입 요소

- 🎨 **방마다 다른 AI 삽화 배경** — 과슈 회화풍으로 통일된 7장의 장면
- 💬 **비주얼 노벨식 스토리** — 클릭할 때마다 타자기 효과로 한 줄씩 나오는 대사, 건너뛰기/다시 보기 지원
- 🎵 **배경음악** — 긴장감 있는 메인 테마와 희망찬 엔딩 테마 (🔊 버튼으로 켜고 끄기)
- 🔓 **효과음** — 자물쇠 해제, 오답 버저, 정답 차임, 문 열림, 종이 넘김
- 📽 **1분 이야기 영상** — 한국어 내레이션과 자막으로 전쟁의 흐름을 정리한 영상이 게임 안에서 바로 재생

## 학습 지원 기능

- 📓 **학습 노트** — 퍼즐을 풀 때마다 핵심 내용이 자동 정리
- 💡 **2단계 힌트** — 사용 횟수 기록 (모둠 대결 시 감점 요소로 활용)
- ⏱ **타이머** 및 탈출 후 **전체 타임라인 정리**

## 에셋 관리

`assets/sources.json`에 생성된 미디어의 원본 URL이 기록되어 있습니다.
GitHub Actions의 **Fetch game assets** 워크플로를 실행하면 목록의 미디어를 내려받아
`assets/`에 커밋합니다 (이미지 JPEG 압축, 오디오 MP3 변환 포함).
