# ⛏ BlockQuest — 게임으로 배우는 우리 반 퀴즈

마인크래프트/로블록스 감성의 **게이미피케이션 에듀테크 플랫폼**입니다.
수업이 끝나면 학생들은 태블릿(갤럭시탭·크롬북·아이패드)으로 코드/QR을 통해 접속해
퀴즈 게임을 풀고, 개인 포인트 랭킹과 학급 전체 포인트에 도전합니다.

## 주요 기능

### 🎮 학생
- **간편 접속**: 6자리 학급 코드 입력 또는 QR 스캔 → 닉네임 + 복셀 아바타 선택 후 바로 입장 (설치·회원가입 불필요)
- **퀴즈 게임**: 정답을 맞히면 블록 파괴 이펙트 + 포인트 획득, 빠르게 맞히면 스피드 보너스, 연속 정답 콤보 보너스
- **랭킹**: 개인 포인트 랭킹(실시간)과 학급 전체 포인트 게이지, 레벨/XP 시스템, 포디움 결과 화면

### 🧑‍🏫 교사 (관리자)
- **AI 퀴즈 자동 생성**: PPT(pptx)·PDF·Excel(xlsx)·Word(docx)·텍스트 파일 업로드, 기사/유튜브 URL, 또는 복사-붙여넣기
  → AI가 자료를 분석해 **학습목표 + 평가 문항(4지선다/OX) + 해설**을 자동 생성
- **퀴즈 검토·수정**: 생성된 문항을 편집한 뒤 원클릭으로 학생들에게 배포(LIVE)
- **실시간 현황**: 학생 × 문항 풀이 매트릭스가 실시간으로 갱신 (Socket.IO)
- **학급 인사이트 시각화**: 문항별 정답률, 학습목표 도달률, 학생 성취 분포(도넛), 학생별 풀이 히트맵, 자동 분석 코멘트

## 실행 방법

```bash
npm install
npm start          # http://localhost:3000
```

### AI 퀴즈 생성 (Claude API)

```bash
export ANTHROPIC_API_KEY=sk-ant-...   # https://platform.claude.com 에서 발급
npm start
```

- API 키가 있으면 **Claude(claude-opus-4-8)** 가 자료를 해독해 학습목표·문항을 생성합니다 (structured outputs로 JSON 스키마 보장).
- API 키가 없으면 **규칙 기반 폴백 생성기**가 동작해 오프라인/데모 환경에서도 전체 흐름을 체험할 수 있습니다.
- 모델 변경: `ANTHROPIC_MODEL` 환경변수 (기본값 `claude-opus-4-8`)

## 사용 흐름

1. 교사: 첫 화면에서 **새 학급 만들기** → 학급 코드/QR 자동 발급
2. 교사: **AI 퀴즈 탭**에서 수업 자료 업로드 → AI가 학습목표·문항 생성 → 검토 후 **퀴즈 시작**
3. 학생: 태블릿에서 코드/QR로 입장 → 퀴즈가 시작되면 자동으로 게임 화면 전환
4. 교사: **실시간 현황**에서 풀이 상황 모니터링 → 종료 후 **인사이트**에서 학급 분석 확인

## 기술 구성

| 영역 | 내용 |
|---|---|
| 서버 | Node.js + Express 5 + Socket.IO (실시간), JSON 파일 영속화 (`data/db.json`) |
| 자료 추출 | pdf-parse(PDF), JSZip(PPTX/DOCX), SheetJS(XLSX/CSV), fetch(기사/유튜브 메타데이터) |
| AI | `@anthropic-ai/sdk` — Claude Messages API + JSON schema structured outputs, 키 없을 시 휴리스틱 폴백 |
| 프론트 | 바닐라 JS + 캔버스 픽셀 텍스처/복셀 아바타/파티클/8비트 사운드(WebAudio), 반응형(태블릿 최적화) |
| 시각화 | 경량 SVG 차트 (접근성 검증 팔레트, 색+기호 이중 표기) |

## 배포 메모

- 데이터는 `data/db.json`에 저장됩니다 (`DATA_DIR` 환경변수로 위치 변경 가능).
- 포트는 `PORT` 환경변수로 변경할 수 있습니다.
- 학생 QR은 `서버주소/?code=학급코드` 로 연결되므로, 배포 시 학생 기기에서 접근 가능한 주소로 서비스해야 합니다.
